# Odin Recipes (With Slightly Too Much Confidence)

Welcome to my very proper recipe website project.

It is simple, static, and held together by HTML, Tailwind CSS, and the kind of optimism usually reserved for British summer weather forecasts.

## What This Project Is

A small multi-page recipe site featuring:
- Garfield's Lasagna
- Peppa Pig's Spaghetti
- Shaun the Sheep's Pizza

Main page: `index.html`  
Recipe pages live in `recipes/`.

## Tech Used

- HTML5
- Tailwind CSS (via CDN)

No frameworks. No build step. No unnecessary drama.

## Project Structure

```text
1-recipes/
  index.html
  images/
  recipes/
    lasagna.html
    spaghetti.html
    pizza.html
  README.md
```

## How To Run

1. Open the project folder.
2. Open `index.html` in your browser.
3. Click a recipe card and pretend you are about to cook any of this.

If you use VS Code, the Live Server extension also works nicely.

## Features

- Hero-style recipe cards on the home page
- Individual recipe pages with:
  - Description
  - Ingredients list
  - Numbered instructions
- Home button on each recipe page
- Gradient backgrounds and responsive-ish layout

## Notes

- Tailwind is loaded from CDN in each page.
- This is a beginner project, so perfection was bravely postponed.
